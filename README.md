# CholloRuta

Aplicación Angular orientada a la geolocalización, la búsqueda de gasolineras y la consulta de rutas mediante APIs externas.

## Funcionalidades principales
- Consulta de información basada en geolocalización.
- Búsqueda de gasolineras.
- Visualización o cálculo de rutas.
- Integración con APIs externas para obtener datos y mostrar resultados en la aplicación.

## Tecnologías utilizadas
- Angular
- TypeScript
- HTML
- CSS
- APIs externas

## Objetivo del proyecto
Este proyecto fue desarrollado como práctica formativa para reforzar el uso de Angular, la integración con APIs y el trabajo con información relacionada con geolocalización y rutas.
